# Assets Folder Guide

Thư mục này chỉ là checklist/tài liệu. Khi code project thật, hãy tạo assets trong `public/images` và `public/fonts`.

## Required Final Assets

```txt
public/images/logo/logo-light.svg
public/images/logo/logo-dark.svg
public/images/banners/home-banner.webp
public/images/banners/our-universe-banner.webp
public/images/products/charm-astra.webp
public/images/products/charm-sirius.webp
public/images/products/charm-polaris.webp
```

## Optional Assets

```txt
public/images/elements/star.svg
public/images/elements/sparkle.svg
public/images/elements/orbit.svg
public/images/about/about-illustration.webp
```

## Font Assets

```txt
public/fonts/youth/
public/fonts/montserrat/
```

Không chia sẻ hoặc commit font nếu chưa có quyền sử dụng rõ ràng.
